# my-cv

> Source repository for my online CV.

The CV is written in Markdown, styled with a custom CSS sheet, and published via **GitHub Pages** as a static site. A PDF export is also available on the live site.

---

## Repository Structure

```
my-cv/
├── .github/
│   └── workflows/        # CI/CD pipelines (GitHub Actions)
└── docs/                 # GitHub Pages source
    ├── index.html        # English version
    ├── index-fr.html     # French version
    ├── style.css         # Custom stylesheet
    ├── assets/
        └── badges/       # Contact badge SVGs
```
